Web Dasar: Basic HTML



Materi oleh: Maresha Caroline

HTML singkatan dari Hyper-text Markup Language. HTML bukanlah bahasa pemograman, melainkan dokumen khusus di mana jika kita mengetik kata khusus (tag) yang diapit dengan '<'' dan '>', browser akan membuat elemen web. HTML bersifat non case-sensitive.

Dalam web, terdapat dua metode utama:
1. Client Side: Berjalan di klien, kode dapat diakses pengguna. Conoh: HTML,CSS,Javascript.
2. Server Side: Berjalan di server, kode tidak dapat diakses pengguna. Contoh: PHP,NodeJS, Ruby, Go.
